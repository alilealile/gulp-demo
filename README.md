# 项目模板

* 开发环境
  - public文件夹
    - js       写的 JavaScript 模块
    - json     用来存放原始数据
    - stylus   stylus 文件
    - css      stylus编译后的常熟市文件
    - img      
    - lib      第三方库 引用的文件
  - view 文件夹 
* 部署环境
  - dist 文件夹
  


* 下载
  - `https://github.com/alilealile/gulp-demo.git`

* 安装依赖
  - `npm install`

* 开发环境
  - 启动
    - `gulp`


* 部署环境
  - 更改 app.js
  
  - 构建
    - `gulp build`

  - 启动
    - `gulp`

* 访问
    - `http://127.0.0.1:8090/chat.html`
